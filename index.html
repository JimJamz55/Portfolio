Here are some of my projects.

The below projects range from personal projects to school assignments.

Custom 3D Printed Hexapod

Custom 3D printed hexapod using RC servos, Odroid XU4 single board computer, custom PIC firmware and custom battery pack.

Simulated Hexapod

Built walking gait for open source hexapod robot design. Simulated robot walk in gazebo.

Sidewalk Segmentation Neural Network

Designed and trained Convolutional Neural Net to perform sidewalk segmentation for the purpose of sidewalk navigation.

Point Cloud Segmentation

Used 3D camera to derive scene point cloud data. Created program to segment point cloud into individual objects.

Adaptive Actuator Controller design

Designed, built, tuned and simulated physics based dynamically robust controller on 2 degree of freedom robot arm.

Sliding Mode Control of a Quadrotor

Used Gazebo simulator to test design of a sliding mode control on quadrotor while tracking quintic trajectory.
